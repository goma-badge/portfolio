PC Manager

MateBook D15では「PC Manager」というソフトウェアがインストールされている
このソフトウェアではPCの最適化を行うことができたり、Huawei製スマートフォンやタブレットとのデータのやり取りができる



また、Huaweiマルチスクリーンコラボレーションという機能により、Huawei製タブレットにノートPCの画面をミラーリングしたり、拡張させてタブレットをサブディスプレイにできる
ミラーリングしたときでもMatePad11の場合はM-Pencilが使用できるので液晶タブレットのように使うこともできる
更に、Huawei製品同士であれば連携させてデータのやり取りも容易にできる

ゲーム性能

MateBook D15に搭載されているRyzenの内蔵グラフィックである「AMD Radeon RX Vega 10」はintel製CPUの内蔵グラフィックより高性能だと言われているため実際に検証した
検証に使用したゲームは「Tom Clancy`s Rainbow Six Siege」というFPSゲームで、設定は最低設定で行った



結果、平均FPSが60FPS 最低FPSが50FPS 最高FPSが73FPSとなった。60FPS出ていると快適に遊ぶことができるので、内蔵グラフィックとしてかなりの好成績であるといえる
検証に使用した「Tom Clancy`s Rainbow Six Siege」は比較的軽めのゲームだが、外出先などでカジュアルにゲームを楽しむ分としては十分であることがわかった
