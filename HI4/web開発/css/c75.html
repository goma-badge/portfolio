Huawei MediaPad M5 pro レビュー

Huawei MediaPad M5 ProはEMUIを搭載したタブレット


EMUI

EMUIとはAndroidをベースとしてカスタマイズされたUI/UX
iPhoneに近いUI/UXやデザインをしているため直感的に利用できる


スペック
価格 ￥50,299(購入当時)
SoC Hisilicon Kirin 960
OS Andorid
メモリ(RAM) 4GB

画面サイズ 10.8インチ
解像度 2560×1600



アプリケーション

MediaPad M5 ProはGMSを搭載しているのでGoogle製アプリをフルに使うことができる。また、AppGalleryも使うこともできる




動画視聴

MediaPad M5 Proの利点として10.8インチの大画面でかつWideVine L1に対応していることがある

Widevineとはスマートフォンなど適用されることの多いDRM(デジタル著作権管理方法)であり、L1 L2 L3までレベルがある。L1だと高画質でストリーミング再生ができる
しかし、WideVine L1に対応していてもAmazon Prime Videoを多くのAndroidタブレットではHD再生することができない
その中でもこのMediaPad M5 ProはAmazon Prime Videoを1080pで再生できる数少ないAndroidタブレットである

以下の写真の左下に「HD 1080p」という表記があるためこのタブレットではAmazon Prime Videoを1080pで再生できていることがわかる
更にダウンロードした映像も1080pで再生が可能である



M-Pen

MediaPad M5 ProではスタイラスペンであるHuawei M-Penを使うことができる
このペンは「Wacom AES」という方式を採用している
USB充電方式でクリップの所をねじると、充電用のUSB-type Cポートが出てくる。また、4096段階の筆圧に対応している
